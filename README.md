## 项目介绍

基于 Element-template 二次开发的后台管理系统。目前主要包括商品管理、订单管理、物流管理、用户账户管理等。

技术栈：vue + vuex + axios + MongoDB + Koa2


## 安装步骤

- clone 项目到本地

- npm install 下载依赖

- npm run dev 运行项目进行开发

- 访问localhost:9528
